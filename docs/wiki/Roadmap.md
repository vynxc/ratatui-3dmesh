# Roadmap

Potential future work:

- Better OBJ normal support and smooth shading.
- Mesh decimation and level-of-detail controls.
- Benchmarks for large models.
- Optional texture sampling into terminal colors.
- Animation helpers.
- Screenshot/GIF generation for docs.
- More public-domain sample assets with attribution.
- Optional `ratatui-core`-only dependency mode if API compatibility permits.
